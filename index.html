Instructions

Make sure SFUI Display fonts are installed See Apple Dev Resources

Make sure nowplaying is put in the folowing location /Users/$USER/nowplaying

put wallpaper folder in /Users/$USER/Pictures/

sub out pics in folders with whatever you want,

SF wallpapers found in screenshots can be found as well as many others here

to remove dock drop this line in terminal

defaults write com.apple.dock autohide -bool true && defaults write com.apple.dock autohide-delay -float 0 && defaults write com.apple.dock autohide-time-modifier -float 0 && killall Dock

Open the glet's with GeekTool

Done
